# Design Brief — ManyTalentsMoney Crypto Dashboard

## Parent Brand
**Many Talents More (MTM)** — the parent company/platform.
The crypto trading dashboard is a SECTION of MTM called **ManyTalentsMoney**.

## Brand Reference
Source logo: `test prep app ManyTalentsMore/app/assets/images/splash-icon.png`

### Key Visual Elements from MTM Logo:
- **Dark background** — deep navy/charcoal, almost black
- **Gold/metallic tones** — rich gold (#c5a03f to #e8c94a range) for primary accents
- **Emerald green** — teal/emerald accent (graduation cap) suggesting growth/success
- **Metallic/embossed feel** — 3D depth, subtle gradients, premium texture
- **Bold typography** — strong, confident letterforms with metallic sheen
- **Serif + sans-serif mix** — "MTP" in bold display, "Many Talents More" in refined serif
- **Badge/shield shape** — the logo sits in a circular/badge frame suggesting trust & authority
- **Tagline bar** — clean subtitle bar beneath the main mark

### What to Carry into ManyTalentsMoney:
- The dark + gold color DNA (premium finance feel — gold = money, wealth)
- The metallic texture feel (subtle gradients, not flat design)
- The confidence of the typography (bold, not timid)
- The dark background (perfect for trading dashboards — Bloomberg/TradingView standard)
- The badge/trust aesthetic (investors need to feel this is serious)

### What to Adapt for Crypto/Finance:
- Replace graduation cap with a finance-relevant accent (chart line, coin, subtle)
- "ManyTalentsMoney" section branding — lighter treatment than the parent logo
- Green for profit, red for loss (standard finance convention) — emerald green from MTM maps naturally to profit color
- Gold accent for highlights, important numbers, call-to-action elements
- Data-dense but organized — more content per screen than the app, but same visual confidence

### Dashboard Should NOT:
- Look like a different company from MTM
- Use flat/minimal Silicon Valley aesthetic (MTM is premium/rich, not minimal)
- Use bright white backgrounds (stays dark)
- Feel like a generic Bootstrap template
- Look like a student/beginner project

### Dashboard SHOULD:
- Feel like opening a Bloomberg terminal if Bloomberg was designed by a luxury brand
- Make someone say "this looks expensive" before reading a single number
- Work beautifully on a phone being shown to someone across a table
- Have the same "trust and authority" feel as the MTM logo
- Make the account value the HERO — big, gold-tinted, live-updating
