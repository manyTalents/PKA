## Lessons

### General: Seek Disconfirming Evidence for Every Diagnosis
- **Category:** diagnostics, methodology
- **Lesson:** After forming an initial diagnostic impression, actively ask "What would make this wrong?" and check the data against that -- anchoring on the first diagnosis causes contradicting evidence to be downweighted.
- **Context:** Known failure mode. Anchoring bias is the most common diagnostic error. The first pattern that jumps out (e.g., "this looks like gut-driven inflammation") can dominate the entire analysis, causing the analyst to unconsciously filter subsequent data through that lens. Deliberately seeking disconfirming evidence is the only reliable antidote.
- **Keywords:** anchoring bias, disconfirming evidence, differential diagnosis, diagnostic error, hypothesis testing

### General: Interpret Trajectories, Not Snapshots
- **Category:** diagnostics, biomarkers
- **Lesson:** A single lab value is a snapshot -- always request prior labs to establish trajectories; a "normal" value that is trending in the wrong direction is more informative than an "abnormal" value that is stable.
- **Context:** Known failure mode. Fasting insulin at 7 looks fine in isolation but is alarming if the trajectory was 4 to 5 to 7 over 18 months. Without prior data, this acceleration is invisible. Always ask for historical results and compare. If no prior labs exist, state the limitation and set a recheck interval.
- **Keywords:** trajectory, trending, snapshot, prior labs, fasting insulin, progression, recheck

### General: Red Flags First, Functional Analysis Second
- **Category:** diagnostics, safety
- **Lesson:** Run the conventional red flag checklist (critical values, urgent findings) FIRST before diving into nuanced functional medicine analysis -- a missed critical value can be life-threatening.
- **Context:** Known failure mode. It is tempting to jump straight into the interesting functional medicine patterns (gut-brain axis, methylation cycles), but a critically elevated liver enzyme or rapidly falling hemoglobin needs immediate escalation. The red flag scan takes 60 seconds and must happen before everything else.
- **Keywords:** red flags, critical values, escalation, safety, liver enzymes, hemoglobin, urgent
